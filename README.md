## Fokus para Produtividade

Este projeto foi desenvolvido com HTML, CSS, Javascript e a criação de um LocalStorage.

Ele tem o objetivo de aumentar a sua produtividade e fazer você focar nas suas tarefas.

Ele permite que usuários:
- Iniciar e pausar sessões de foco;
- Alternar entre foco, descanso curto e descanso longo;
- Visualizar a contagem regressiva em tempo real;
- Escrever as tarefas à serem feitas ilimitadamente;
- Ouvir uma música enquanto estuda.
